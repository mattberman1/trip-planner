# Trip Planner

A SwiftUI app for planning trips using SwiftData.

## Requirements
- Xcode 15 or later
- iOS 17 SDK

## Building
Open `TripPlanner.xcodeproj` in Xcode and run the `TripPlannerApp` scheme.

## Running Tests
Use the Product \> Test menu in Xcode or run `xcodebuild test` on the command line.
